# 📸 Folder Galeri - Pondok Es Teller ZR

## Cara Upload Foto Asli Anda

### **Foto yang Dibutuhkan:**

Rename dan upload foto Anda dengan nama berikut:

1. **kedai-depan.jpg** - Tampak depan kedai yang ramai pelanggan
2. **buah-segar.jpg** - Buah-buahan segar colorful (pepaya, nanas, buah naga, dll)
3. **suasana-dalam.jpg** - Suasana dalam kedai dengan pelanggan
4. **display-esteller.jpg** - Display es teller dengan alpukat
5. **booth-mall.jpg** - Booth di mall
6. **alpukat-segar.jpg** - Alpukat segar melimpah
7. **menu-board.jpg** - Menu board / banner
8. **counter-pembuatan.jpg** - Counter pembuatan / proses

### **Spesifikasi Foto:**

- **Format**: JPG atau PNG
- **Ukuran**: Maksimal 2MB per foto
- **Resolusi**: Minimal 800x600px (landscape)
- **Aspect Ratio**: 4:3 atau 16:9 (recommended)

### **Cara Upload:**

#### **Option 1: Via GitHub (Recommended)**
1. Buka: https://github.com/rannn2008/estellerzrpadang
2. Navigate ke: `webzrpromosi/foto2/galeri/`
3. Klik "Add file" → "Upload files"
4. Drag & drop foto Anda
5. Rename sesuai list di atas
6. Commit changes

#### **Option 2: Via Local**
1. Copy foto Anda ke folder ini
2. Rename sesuai list di atas
3. Commit & push ke GitHub:
   ```bash
   git add foto2/galeri/
   git commit -m "feat: Add real gallery photos"
   git push origin main
   ```

### **Fallback:**

Jika foto belum di-upload, website akan otomatis menggunakan:
- Foto placeholder dari Unsplash (sementara)
- Foto yang sudah ada di `foto2/estelleroriginal.jpg`

### **Setelah Upload:**

Website akan otomatis update dalam 2-3 menit (Vercel auto-deploy).
