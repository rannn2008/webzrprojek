## 🧠 Teknologi Pintar

### **Fuzzy Matching**
Chatbot bisa mengenali variasi pertanyaan:
```
"Menu apa aja?" = "Ada menu apa?" = "Jual apa?" = "Pilihan menu?"
→ Semua dijawab dengan benar!
```

### **Context-Aware**
Chatbot tahu konteks dari data real-time:
```
User: "Menu termurah?"
Bot: "Menu termurah kami: Es Teller Original - Rp12.000"
     (Data dari database real-time!)
```

### **Smart Scoring**
Chatbot memilih jawaban terbaik berdasarkan keyword matching:
```
"Harga es teller alpukat berapa?"
→ Keyword: "harga" (5 poin) + "alpukat" (7 poin) = 12 poin
→ Match dengan kategori "price" + "menu"
→ Jawaban akurat!
```

### **Dynamic Responses**
Jawaban berubah sesuai kondisi:
```
Jam 11:00 WIB:
User: "Buka?"
Bot: "Kami buka setiap hari jam 10.00 - 22.00 WIB
      Sekarang kami BUKA! Mampir langsung atau pesan via WhatsApp ya!"

Jam 23:00 WIB:
User: "Buka?"
Bot: "Kami buka setiap hari jam 10.00 - 22.00 WIB
      Maaf, sekarang kami TUTUP. Tapi kamu bisa pesan dulu via WhatsApp untuk besok!"
```

### **Random Responses**
Jawaban bervariasi agar tidak monoton:
```
User: "Terima kasih"
Bot (Random):
- "Sama-sama! Jangan lupa pesan Es Teller ZR ya. Dijamin segar! 🧊"
- "Senang bisa membantu! Kapan-kapan mampir ya ke toko kami! 😊"
- "Sama-sama! Kalau ada pertanyaan lagi, jangan sungkan tanya ya! 💚"
- "You're welcome! Semoga hari kamu menyenangkan! ☀️"
```

### **Class: SmartChatbot**
```javascript
class SmartChatbot {
    constructor(menuData)
    initializeKnowledgeBase()
    findBestMatch(question)
    generateResponse(match)
    getFallbackResponse()
    chat(question)
    formatPrice(price)
    updateMenuData(newMenuData)
    getHistory()
    clearHistory()
}
```

### **Knowledge Base Structure**
```javascript
{
    categoryName: {
        keywords: ['keyword1', 'keyword2', ...],
        responses: ['response1', 'response2', ...],
        type: 'random' | 'function'
    }
}
```

### **Integration Points**
1. **Initialization**: After menu data loaded
2. **Update**: When menu data changes
3. **Chat**: User sends message
4. **Response**: Bot generates smart response
